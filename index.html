HTML (Hyper-Text Markup Language)

HYML is a programming language that is made up of "elememts." Each element usually contains an opening and closing "tag" that determines the way an element's content behaves. This table will list some common tags found in HTML.

Tag Name: Format: Tag Description/Notes: Example:
Paragraph <p> A paragraph tag allows you to add lines of text to your HTML document. <p>Here's some text. This is what it looks like.</p>
Header <h1> A header tag adds special header text to your HTML document. h1 - h6 is formated largest to smallest. <h1>Headers help organize your content on page.</h1>
Button <button> If you want a button on screen, you can use the button tags. <button>Add some engaging text here!</button>
Image <img/> The image tag is used to add images to your HTML document. Note that the image tag doesn't have closing tag. This is called a self-closing tag. <img src="image-location-link.jpg"/>
Unordered List <ul> You can make a bullet point list using this type of list tag. The list item tags (li) define each item in your list. <ul> <li> List content... <li/> <ul>
Ordered List <ol> You can make a numbered list using this type of list tag. The list item tags (li) define each item in your list as well. <ol> <li> List content... <li/> <ol>
